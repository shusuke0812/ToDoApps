ToDoアプリづくりを通して様々な言語の勉強をする
====
## 言語一覧
- JavaScript
- TypeScript
- Swift
- Kotlin
- Ruby
- Python

## 開発環境
- MacOS Catalina version 10.15
  
  #### Swift  
  - Xcode version 11.3
  - pod version 1.9  
  - swift 5.1  

  #### Node
  - npm verision 6.13
  - node verison 12.16  

  #### Python
  - pip version 20.1
  - python version 3.8

## アプリ機能
- todoをテキスト入力して、リストで表示する
- 登録したtodoリストの横に完了ボタンを置き、ボタンを押したらtodo項目に取り消し線を入れる
